Stitch - Powerpoint Merging Tool

HackZurich Fall 2015

This was done as a hackathon project during HackZurich 2015. Our team created a tool using the Dropbox API that enabled multiple powerpoint presentations to be stitched together into one seamless presentation. This solves the problem of having to close out, find the file, and reopen a new file in between a bunch of successive presentations. Users just upload all of their presentations to a Dropbox repository, and then the script runs and outputs a single file. Our team won the Dropbox API prize for this project.
